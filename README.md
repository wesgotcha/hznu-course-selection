# 🎓 HZNU 2025 级多专业本科培养方案选课 & 毕业达标管理系统

这是一个针对**杭州师范大学 2025 级多专业本科培养方案**（目前已支持：**计算机科学与技术**、**软件工程**、**电子信息工程**、**地理信息科学**、**计算机科学与技术（师范）**）量身定制的课程进度与毕业达标管理工具。系统采用纯静态网页技术架构，支持本地持久化存储，并在不同专业之间进行了数据隔离与定制化规则判定。无需任何服务器即可本地双击流畅运行，旨在帮助同学们清晰规划修读路径，实现一键算分、直观对比、自主查漏补缺。

## 🌟 核心功能特性

### 1. 现代化的三栏式响应式面板
* **第一栏 (主操作区)**：
  - **模糊检索**：支持搜索课程名称、代码、建议学期和备注信息。
  - **双重过滤器**：按“课程类别”和“标准学期”进行交叉精确过滤。
  - **课程表单**：展现完整的培养方案要求必修与选修课程列表。
* **第二栏 (选课看板与规则校验 - 可折叠)**：
  - **KPI 控制台**：实时追踪“总修读学分”（目标 151 学分）、“已达标规则数”、“剩余应修学分”及“已勾选课程数”。
  - **学期学分网格**：自动按学期汇总呈现已规划学分负载（例如：`大一上 24.5学分`）。
  - **培养方案规则树**：完整解析并动态展现 8 大类目标学分及达标进度（红/绿条形进度条）。
* **第三栏 (选课清单 - 可折叠)**：
  - 将选中的课程自动提取为侧边选课单。
  - 自动按“计划修读学期”进行分组排序，展示每一学期修读的课程明细与单学期学分小计。

### 2. 智能学期覆盖范围映射 (Term Coverage Map)
- 学期过滤器固定为 8 大标准学期（大一上至大四下）。
- 后台引入了**建议修读学期覆盖范围算法**：
  - 当您过滤“大一上”时，系统将智能包含如“大一上”、“大一上/大一下”（一秋/一春滚动）以及“大一至大四”（春秋滚动）等所有在该学期滚动开设的课程，确保滚动课不漏选、不误判。

### 3. 通识选修自主添加与模板快捷绑定
- **自主添加通识课**：点击“添加通识课”主按钮，可任意多次添加各类真实的通识课程（支持设定课程名、学分、所属类型、计划修读学期）。
- **行内快捷添加模板**：保留列表中的 4 个核心通识类别模板引导行，名称旁配有小巧的 `+ 添加` 按钮。点击后，弹窗表单中“通识课程类型”会自动锁定为该行所属类型，用户仅需输入课程名与学分即可快速录入。
- **细化子规则校验**：自动校验毕业必修的三个细化通识选修学分约束：
  - 艺术鉴赏与审美体验（需 `≥ 2` 学分）
  - 人工智能与现代生活（需 `≥ 2` 学分）
  - “四史教育”专题（需 `≥ 1` 学分）
- **数据弹性兼容**：计算子要求学分时，系统会自动将**用户录入的真实通识课学分**与**直接勾选的对应占位课模板学分**进行合并统计，不强求用户录入明细。

### 4. 绑定专业的规则引擎与多模块判定
- 针对不同专业的独特方向选修模块要求，系统内置了专业的规则数据库与判定引擎（根据专业自动加载）：
  - **多选一方向模块** (`selectOne` 规则，适用于计算机、软件工程、电子信息等）：系统会自动挑选您当前已修读学分最高的方向模块作为主攻方向，并实时判断是否达到该方向的毕业门槛（如同一模块 `≥ 7.5` 或 `10` 学分）。
  - **全必修模块最低要求** (`allRequired` 规则，适用于地理信息科学等）：六大模块（公共基础、GIS、遥感、测绘、地学、人工智能）必须**分别**满足各自的最低学分要求，系统将它们自动拆分为并列的毕业指标，分别显示达标状态，实现差异化的规则评估。

### 5. 进度备份与极佳的用户体验 (UX)
- **本地持久化**：使用浏览器 `localStorage`，页面刷新或断网数据绝不丢失。
- **导入与导出**：一键导出 JSON 进度包文件，支持在不同设备/浏览器中导入恢复进度。
- **暗黑/明亮主题切换**：支持跟随系统偏好，并可通过右上角太阳/月亮按钮一键无缝平滑切换。
- **自定义确认框**：清空操作使用半透明磨砂玻璃遮罩的自定义对话框替代浏览器原生 `confirm`，交互体验更显高级。
- **一键打印**：内置打印样式表，打印时自动隐藏全部搜索控制按钮，完美呈现整洁的纸质选课报告。

## 🛠️ 技术架构

本系统是典型的 **BaaS (Backend as a Service) 静态网页架构**，所有逻辑和样式都在客户端浏览器运行：
* **核心结构**：HTML5 语义化标签。
* **排版系统**：CSS Grid（三栏布局） + Flexbox 弹性流式排版，引入 Google Fonts（Inter & Outfit）提升文字品质。
* **动态逻辑**：纯 JavaScript 实现，采用数据驱动渲染和声明式聚合计算。
